Développement d’API REST : Construire, Tester, Dépanner

Apprenez à créer, configurer, tester et déboguer des API REST à l’aide d’exemples pratiques

Les API REST constituent la colonne vertébrale des applications et services web modernes. Dans ce cours, vous apprendrez à concevoir, développer, tester et dépanner des API REST étape par étape à l’aide d’outils pratiques et d’exemples du monde réel.

En commençant par les bases de HTTP et de JSON, vous construirez une API REST entièrement fonctionnelle, comprendrez comment fonctionnent les différentes méthodes HTTP et apprendrez à tester votre API avec Postman.

Ce que vous allez apprendre

  • Les fondamentaux de HTTP et de l’architecture des API REST
  • Comment les API REST communiquent à l’aide de JSON
  • Comment configurer et créer un projet d’API REST
  • Créer des modèles et des contrôleurs pour les endpoints de l’API
  • Comprendre les méthodes GET, POST, PUT, DELETE et PATCH
  • Créer des routes personnalisées et filtrer les résultats de l’API
  • Empêcher les enregistrements en double dans les opérations POST
  • Tester les API avec Postman
  • Automatiser les tests d’API et utiliser des variables dans Postman
  • Dépanner les problèmes courants des API REST

Sections du cours

  • Attentes du cours et aperçu général
  • Téléchargement et installation du logiciel de bureau
  • Fondamentaux des API REST

  • Création de modèles et de contrôleurs d’API

  • Filtrage avancé des API et règles métier
  • Tests d’API REST avec Postman
  • Tests automatisés des API
  • Dépannage et débogage des API
